Data Entry form preparation

To entry access data entry, select Data entry from menu. To be able to enter data, first you must fill data entry preparation form. This including selection of organisation unit, data entry form and period.

Types of Data Entry form

Basically two types are supported default and section. Default data entry form is just list of all data elements as questions. Section is customized data entry form contains grouped related data elements, it is set on DHIS 2 server.

Data Entry form form configurations

It including setting label for data entry form. It can be either DHIS 2 form name or DHIS 2 display name.To configure, select Setting from menu, go to "Form Label or Question title setting" sections and select appropriate label you prefer.

Report Generation process

To generate reports on mobile devices, select Html Reports from menu, select report form list of reports, fill all report parameter if present.

Supported reports

So far it support html reports designed to use data values saved on offline storage on mobile device using date entry form of this app.

To access setting, select Setting from menu

Synchronizations

Synchronization time is time to take for an app to check data values in offline storage on mobile device to be sent to login server when you have network connections. To set Synchronization time, first enter value and then select time type(seconds, minutes or hours) based on your need

Form label and questions title

This part allow you to select label or question title for data entry form. It can can be either DHIS 2 form name or DHIS 2 display name based on your preference.

data reset

App gives you two option buttons to reset or clear data stored on local/offline storage, Reset Data Entry values and Clear/Reset Offline storage.Reset Data Entry values is for clear all data values entered and clear data stored on local/offline storage option for clear or delete all data stores in the mobile device for an app
